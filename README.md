# 🐦 Twitter Clone

A fully responsive Twitter Clone built using HTML, CSS, and Tailwind CSS.  
This project recreates the modern Twitter UI with responsive layouts, sidebar navigation.

---

# 📌 Features

- ✅ Responsive Twitter UI
- ✅ Built with Tailwind CSS
- ✅ Sidebar Navigation
- ✅ Tweet Feed Layout
- ✅ Profile Widgets
- ✅ Mobile Friendly Design
- ✅ Clean & Organized Code Structure
- ✅ Modern UI Components

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- Tailwind CSS
